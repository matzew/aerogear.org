registerWithClientInfo:success:failure:

Registers your mobile device to the AeroGear Push server so it can start receiving messages.

- (void)registerWithClientInfo:(id)clientInfo success:(id)success failure:(id)failure

Parameters

clientInfo

A block object which passes in an implementation of the AGClientDeviceInformation protocol that holds configuration metadata that would be posted to the server during the registration process.

success

A block object to be executed when the registration operation finishes successfully. This block has no return value.

failure

A block object to be executed when the registration operation finishes unsuccessfully. This block has no return value and takes one argument: The NSError object describing the error that occurred during the registration process.

Discussion

Registers your mobile device to the AeroGear Push server so it can start receiving messages.
