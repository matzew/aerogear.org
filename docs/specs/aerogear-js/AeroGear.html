Status: Experimental
The AeroGear.ajax is used to perform Ajax requests.
Parameters:
Name Type Argument Default Description
settings Object <optional>
 {} the settings to configure the request
Properties
Name Type Argument Default Description
url String <optional>
 the url to send the request to
type String <optional>
 "GET" the type of the request
dataType String <optional>
 "json" the data type of the recipient's response
accept String <optional>
 "application/json" the media types which are acceptable for the recipient's response
contentType String <optional>
 "application/json" the media type of the entity-body sent to the recipient
headers Object <optional>
 the HTTP request headers
options.success AeroGear~successCallback <optional>
 callback to be executed if the AJAX request results in success
options.error AeroGear~errorCallback <optional>
 callback to be executed if the AJAX request results in error
settings.params Object <optional>
 contains query parameters to be added in URL in case of GET request or in request body in case of POST and application/x-www-form-urlencoded content type
settings.data Object <optional>
 the data to be sent to the recipient
